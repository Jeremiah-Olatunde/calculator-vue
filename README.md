# CALCULATOR VUE

### ABOUT
A simple calculator application built using VueJS.  
**_NOTE_** : Intended for pedagogical/pedantry purposes.

### FUNCTIONS SUPPORTED _v(1.0.0)_
* Addition _(+)_
* Subtraction _(-)_
* Multiplication _(*)_
* Division _(/)_
